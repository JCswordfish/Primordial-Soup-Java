# Primordial Soup in Java

Below contains a rundown of each file's purpose:

## Front-end
- `Screen.java` - Manages the screen that displays the board. Everyone takes turns using the same screen.

- `Runner.java` - Runs the game.

## The Game:

- `Game.java` - Manages the workings of the game. 

- `Environment.java`, `Amoeba.java`, `AdvancedGene.java`, `Food.java`, `Gene.java`, `Species.java` - Various game objects.
